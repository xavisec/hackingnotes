# Hacking Notes  

## Configuration files: 
|Description                | Command                         |Syntax                 |
|----------------|-------------------------------|-----------------------------|
| Use of VPN | openvpn           |openvpn [FILE]            |
| Use of nmap | nmap           | nmap [IP]/[RANGE] [OPTIONS]            |
| UDP nmap scan | nmap           | nmap [IP]/[RANGE] -sU             |
| TCP nmap scan (no ping) | nmap           | nmap [IP]/[RANGE] -Pn             |
